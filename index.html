Installation

Installez Kronobus sur votre téléphone

Aucune boutique d'applications. Directement depuis votre navigateur, en 3 étapes.

1

Ouvrez kronobus.app dans votre navigateur

Safari sur iPhone, Chrome sur Android — les deux fonctionnent.

2

Appuyez sur « Ajouter à l'écran d'accueil »

iPhone : bouton partage → Ajouter. Android : menu → Installer.

3

Ouvrez l'app depuis votre écran d'accueil

Kronobus s'ouvre comme une vraie app, sans barre de navigation, même hors-ligne.

📱
iPhone
iOS 14+ · Safari requis · Icône sur l'écran d'accueil
🤖
Android
Android 8+ · Chrome requis · Installation native
💻
Desktop
Windows, Mac · Chrome ou Edge · Raccourci bureau
⚡
Hors-ligne
Fonctionne sans connexion. Sync automatique au retour.
