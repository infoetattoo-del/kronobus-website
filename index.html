Roulette heure/minute avec défilement infini. Plusieurs services par jour. Amplitude automatique.

Installation

Installez Kronobus sur votre appareil

Sans passer par l'App Store. En 3 gestes, l'application est sur votre écran d'accueil — comme n'importe quelle app.

1
🌐

Ouvrez Safari

Allez sur kronobus.app dans Safari — pas Chrome, pas Firefox. Safari uniquement sur iPhone.

2
⬆️

Appuyez sur le bouton Partager

Le bouton carré avec une flèche ↑ en bas de l'écran.

3
➕

Ajouter à l'écran d'accueil

Faites défiler, appuyez sur « Ajouter à l'écran d'accueil » puis confirmez.

✓ Kronobus s'ouvre maintenant en plein écran, sans barre Safari, comme une vraie app.
1
🌐

Ouvrez Chrome

Allez sur kronobus.app dans Google Chrome.

2
⋮

Menu → Ajouter à l'écran d'accueil

Appuyez sur ⋮ puis « Installer l'application ».

3
✅

Confirmez l'installation

Une fenêtre s'ouvre. Appuyez sur « Installer ».

✓ Sur Android, l'expérience est identique au Play Store — barre de téléchargement incluse.
1
🌐

Ouvrez Chrome ou Edge

Allez sur kronobus.app dans Chrome ou Edge.

2
💾

Cliquez sur l'icône d'installation

Une icône apparaît dans la barre d'adresse. Ou Menu → Installer Kronobus.

3
🖥️

Lancez depuis le bureau

Un raccourci apparaît sur votre bureau. Kronobus s'ouvre sans navigateur.

✓ Kronobus fonctionne comme une application Windows ou Mac classique.

Installation

Installez Kronobus sur votre téléphone

2

3

Kronobus s'ouvre comme une vraie app, sans barre de navigation, même hors-ligne.
